Управление в игре:
В игре:
w, a, d, ar_l, ar_up, ar_r - управление персонажем
[1, 8] - изменить блок для постановки
lclick - удалить блок
rclick - поставить блок
В меню:
wclick по уровню создать новый уровень
--------------------------------------------------------------------------------------------------------------------
Инструкция для чайников:
1) При в ходе мы видем главное меню
   a) Выбираем один из пяти уровней
   б) Если нажать правую кнопку мыши по уровню, он создатся заново
2) В игре мы можем ходить на wad
   Ставить блоки 
   Удалять блоки
3) Если мы хотим выйти то, нажимаем на "шахматное поле"
4) Смотрим результаты
   И ещё раз нажимаем на шахматное поле
