О проекте:

   pygame
   numpy
   sys

   random

--------------------------------------------------------------------------------------------------------------------

Использование:

1) Установка необходимых библиотек.
    Выполните следующие команды в терминале python:
    pip install pygame
    pip install numpy
    Остальные библиотеки встроеные

2) Запуск и игра.
    При запуске игры в главном меню Вы увидите название игры и выбор мира (1 из 5)
    Движение на a, d и пробел, либо стрелки влево, вправо и вверх
    Ломать и ставить блоки на левую и правую кнопки мыши соответственно
    Для выбора блоков используйте кнопки 1 - 8
    Подсчёт результатов на "шахматное поле"
    Для выхода повторно нажимаем на "шахматное поле"
    Для пересоздания мира ПКМ по нему в главном меню
